# ApexPlanet Data Analytics Project

## Project Overview
This project performs basic data analysis and visualization using Python, Pandas, and Matplotlib. It loads an Excel dataset, explores the data, calculates key statistics, and generates visualizations to understand sales performance and customer demographics.

## Features
- Load Excel dataset
- Display first 5 records
- Check missing values
- View dataset information
- Generate statistical summary
- Calculate total sales
- Calculate average customer age
- Analyze sales by category
- Analyze sales by city
- Create bar chart for category-wise sales
- Create pie chart for category-wise sales
- Create histogram for age distribution

## Technologies Used
- Python
- Pandas
- Matplotlib
